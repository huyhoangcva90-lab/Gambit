# Card images

Đặt ảnh lá bài đã hoàn thiện trong thư mục này theo định dạng WebP.

Quy ước tên file:

```text
the-fool.webp
the-magician.webp
the-high-priestess.webp
...
the-world.webp
```

Danh sách ID đầy đủ nằm trong `../../data/cards.js`.

Khuyến nghị:

- Tỷ lệ ảnh: `2:3`
- Kích thước tối thiểu: `800 x 1200 px`
- Dùng cùng một hệ màu và bố cục cho toàn bộ bộ bài
- Tối ưu mỗi ảnh dưới khoảng `300 KB`
- Nếu ảnh chưa tồn tại, ứng dụng tự hiển thị tên lá bài thay thế
